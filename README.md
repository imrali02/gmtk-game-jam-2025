## Controls

- Movement: WASD
- Dash: Spacebar
- Rewind: R

## Project Structure

The project follows a topic-based organization:

```
/
├── entities/           # Game entities
├── levels/             # Game levels/scenes
├── ui/                 # User interface elements
├── globals/            # Global scripts
└── assets/             # Textures and sprites
```
